# ZeroDrag

A Next.js application starter with authentication, payments, and AI infrastructure.

## What it solves

Provides production-ready foundations for web applications that need user accounts, billing, and feature gating.

## Who it's for

Experienced developers building applications that require:

- User authentication and sessions
- Subscription billing with multiple payment providers
- Feature access control
- Email communications
- Optional AI integration
